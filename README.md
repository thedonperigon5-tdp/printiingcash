# Quiz Website Clone

This directory contains a complete clone of the quiz website from:
https://quiz.getrelatio.com/Pe3MbPt/aa3rhk1i-wnig-yqv7-olp5lqqzyvfz/

## Files Cloned

### HTML
- `index.html` - Main quiz page (step 2 of 26)

### CSS Files
- `css/reset.css` - CSS reset styles
- `themes/pe-black/css/style.css` - Main theme styles
- `thirdparty/slick/slick.css` - Slick slider styles
- `funnels/Pe3MbPt/css/style.css` - Funnel-specific styles

### JavaScript Files
- `js/jquery.min.js` - jQuery library
- `js/default.js` - Main quiz functionality
- `thirdparty/slick/slick.js` - Slick slider library
- `js/event_tracking_manager/fb_manager.js` - Facebook event tracking

### Images
- `themes/pe-black/images/Relatio.png` - Logo
- `funnels/Pe3MbPt/images/103_1.png` - Body type image (Magro)
- `funnels/Pe3MbPt/images/103_2.png` - Body type image (Normal)
- `funnels/Pe3MbPt/images/103_3.png` - Body type image (Grande)

## Features

This quiz page includes:
- Dark theme design
- Progress bar (step 2 of 26)
- Interactive image radio buttons for body type selection
- Facebook Pixel tracking
- Amplitude analytics
- Responsive mobile-first design

## To View

Simply open `index.html` in a web browser. Note that some external tracking scripts (Facebook, Amplitude) are still referenced from their original CDN sources.
